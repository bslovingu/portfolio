#서울시 공용체육시설 예약 도우미 #SPORnSER

프로젝트 설명

"SPORnSER"는 갑자기 약속을 잡아서 함께 운동할 운동친구들이 없는 현대인을 위한 체육시설 예약을 돕는 웹 어플리케이션입니다. 사용자의 현재위치와 인근 체육시설위치를 보여주며, 경로안내 기능이 있습니다. 지역구/종목별 조회하여 선택한 체육시설종목을 예약/예약취소 할 수 있으며, 저장된 예약 데이터를 지역구/종목별 이용현황으로 시각화하여 보여줍니다. 공지사항, 신고게시판(+댓글)과 같은 커뮤니티의 기능 역시 가지고 있습니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터 : 서울시 공공체육시설 현황
- 기타 : Chart.js, kakao 지도 API, Bootstrap,
            jQuery, JavaScript

#이슈_한눈에_보기 #It's u, Issue!

프로젝트 설명

'It’s u, Issue!'는 크롤링을 통하여 네이버 급상승 검색어와 각 검색어별 연관검색어를 1차적으로 제공합니다. 검색어 클릭 시, 뉴스의 최근자료를 분류하며 R과 amCharts를 통하여 뉴스본문과 댓글을 분석 및 시각화합니다. 뉴스본문은 단어의 빈도수를 분석하여 워드클라우드로 시각화된 자료를 제공하며, 뉴스댓글은 오피니언마이닝을 거쳐 부정/긍정 정도를 시각화된 자료로 제공합니다. JSoup 라이브러리 사용시 알 수 없던 뉴스댓글의 전체 개수를 URL connection과 구현한 로직으로 구하며, 일부를 추출(표본추출)한 표본집단을 활용하여 긍정/부정 정도의 오차범위를 제공합니다. 또한, Google Trends를 응용하여 사용자가 클릭한 검색어의 시간대별/지역별 국내 검색현황을 제공합니다. 그 밖의 기능으로는 회원가입, 로그인, 아이디 저장하기, 아이디 찾기, 비밀번호 메일 재발급, 마이페이지(정보수정/회원탈퇴)와 문의사항 메일발송이 사용자 기능으로 있으며, 관리자 기능으로 마이페이지(정보수정/회원탈퇴)와 회원관리(회원 조건 검색/정보수정/회원탈퇴/권한변경)가 있습니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : MySQL, MongoDB
- 기타 : amCharts, Google Trends, aws EC2,             jQuery, JavaScript, Bootstrap
